Go Severless

Objective

Go Serverless is an effort to build highly available, scalable, serverless website using Amazon S3 bucket. This website is designed to handle sudden traffic spikes and grow unlimitedly without human intervention.

Highlights

  • No server is used to host this website. It is hosted in Amazon S3 bucket.
  • Source files are stored in Github public repository.
  • Deployment to Amazon S3 bucket is automated using CircleCI which is integrated with Github.
  • Google Analytics tracking code collects data about of user interaction with the website and sends weekly reports to Admin.
  • Search Engine Optimization is performed.
